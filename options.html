Options

You can use { name } syntax for placeholders in the alert box message.

{ which }
"one window" or "total", depending on where there were too many tabs. If both, uses "total".
{ maxWhich }
maxTotal or maxWindow, depending on where there were too many tabs. If both, uses "total".
{ maxTotal }
Max. number of total tabs
{ maxWindow }
Max. number of tabs in one window

Change values to update them
